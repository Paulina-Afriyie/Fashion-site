# 👗 Paulina's Fashion Store Website

A modern and responsive **Fashion Store Website** built using **HTML, CSS, and JavaScript**.
This project showcases a clean e-commerce layout with product display, shopping cart functionality, and user-friendly design.

---

## 🚀 Features

* Responsive design (mobile & desktop)
* Modern homepage with hero section
* Product grid layout
* Shopping cart system
* Contact page with form and map
* Professional UI/UX design
* Login page interface

---

## 📖 Project Background

Paulina's Fashion is a web-based fashion store designed to provide users with a convenient way to browse and interact with clothing products online.

The goal of this project is to create a visually appealing and user-friendly platform where customers can explore fashion items, view prices, and simulate online shopping.

This project demonstrates key web development concepts such as responsiveness, layout design, and basic interactivity using JavaScript.
